# Outreach для фотостудий

Цель: собрать до 1000 публичных контактов фотостудий по России и отправить аккуратное B2B-письмо с предложением Virtual AI Photo Studio, промокодом и ссылкой на блок "до/после".

## Рекомендуемый запуск

1. Создать отдельный почтовый ящик на домене, например `partners@virtualphotostudio.ru`.
2. Настроить SPF, DKIM и DMARC у домена.
3. Взять 1 город или регион на тест: например Москва, Санкт-Петербург, Екатеринбург.
4. Собрать первые 50 контактов, проверить глазами качество базы.
5. Отправлять не более 20-40 писем в день с нового ящика.
6. В письме обязательно оставить мягкий отказ: "Если неактуально, ответьте 'стоп', больше не напишем".

## Источники контактов

Лучший вариант для n8n:

- Google Places API: поиск по запросу `фотостудия Москва`, `photo studio Moscow`, `аренда фотостудии Москва`.
- Apify Google Maps Scraper: проще для массового сбора, но платный/лимитированный.
- 2GIS/Яндекс вручную или через разрешённый API, если есть доступ.

Не рекомендую агрессивно парсить сайты без лимитов: это быстро ломается и может привести к блокировкам.

## Поля лида

Минимальный CSV:

```csv
unique_key,studio_name,city,website,email,phone,source,promo_code,status,last_contacted_at,raw
email:info@example.com,Studio Name,Москва,https://example.com,info@example.com,+79990000000,google_places,STUDIO,new,,"{}"
```

В продуктовой версии эти же данные сохраняются в Supabase-таблицу `outreach_leads`.

## Сегментация

Для первой волны:

- фотостудии с интерьерными залами;
- студии с арендой залов для фотографов;
- студии, у которых есть сайт и email;
- города-миллионники.

Не отправлять:

- фотографам-фрилансерам без студии;
- свадебным агентствам;
- магазинам техники;
- контактам без публичного business email.

## Промокод

Для тестовой рассылки можно использовать:

- `STUDIO`: 2 промо-запуска по 20 фото для фотостудий.
- `FRIEND`: 3 промо-запуска по 20 фото.

Для рассылки по фотостудиям используем отдельный код `STUDIO`, чтобы видеть именно этот канал.
